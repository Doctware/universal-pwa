A stunning, framework-agnostic JavaScript SDK for seamless Progressive Web App installations. Zero dependencies. Endless possibilities.

Features

  • Universal Compatibility: Works natively with Vanilla JS, React, Vue, Svelte, or Next.js.
  • Platform Intelligent: Auto-detects iOS, Android, macOS, and Windows installation patterns.
  • Shadow DOM UI: Guaranteed isolation. Your CSS will never break our prompt, and vice versa.
  • Smart Cooldowns: Built-in dismissal tracking so you don't annoy your users.

Installation

Install via NPM for modern bundlers:

npm install universal-pwa

Or use it directly in the browser via CDN:

<script src="https://unpkg.com/universal-pwa@latest/dist/index.umd.js"></script>

Usage Guide

Initialize the SDK once in your application's root:

// Import if using NPM, otherwise use the global UniversalPWA object
import { init } from 'universal-pwa';

const { engine, ui } = init({
  appName: 'My Amazing App',
  themeColor: '#6366f1',
  description: 'Install this app for offline access and native performance!',
  position: 'bottom', // 'top', 'bottom', or 'center'
  delay: 3000, // Wait 3 seconds before prompting automatically
  dismissCooldownDays: 7 // Respect user dismissals
});

// You can also manually trigger the prompt anytime
document.getElementById('install-btn').addEventListener('click', () => {
  ui.show();
});

React Support

We ship with native React hooks and components to make integration even easier.

import { usePWAInstall, PWAInstallButton } from 'universal-pwa/react';

function App() {
  const { isInstallable, showPrompt } = usePWAInstall();

  return (
    <div>
      {isInstallable && <PWAInstallButton config={{ appName: 'React App' }} />}
    </div>
  );
}
